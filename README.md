# PSPDFKitOCR

Convert scanned PDF into machine readable documents.

PSPDFKitOCR quickly converts scanned documents into machine-readable text, unlocking full PDF functionality for the entire document.

## Installation

```
pod 'PSPDFKitOCR'
```
